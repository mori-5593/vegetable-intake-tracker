## サービス概要
視聴したアニメの感想を記録し、自分の「見た・見たい」を管理するアニメ特化型ログサービスです。
友人の投稿を通じて新しいアニメを発見し、感想を共有できます。
将来的には外部API連携により、正確な作品情報を簡単に取得・記録できる機能も追加予定です。
## このサービスへの思い・作りたい理由
私自身、アニメが大好きで多くの作品を見てきましたが、友人から「おすすめを教えて」と言われた際にタイトルがパッと思い出せないことが多々ありました。
また、友人が今期のアニメを必死にメモしている姿を見て、もっと手軽に、かつ楽しく記録を残せる場所があればいいのにと感じたのがきっかけです。

自分の思い出を整理するだけでなく、友人との会話の種になるようなサービスを作りたいと考えています。

## ユーザー層について
・アニメ視聴が趣味の方: 自分の視聴履歴を可視化して楽しみたい層。
・次に見るアニメを探している方: 友人や他人のリアルな熱量を感じる感想を参考にしたい層。
・記録を習慣化したい方: 複雑なツールではなく、手軽に「見た・見たい」を管理したい層。

## サービスの利用イメージ
ユーザーは、見たアニメの感想や5段階評価を投稿します。
友人の投稿を見て「面白そう」と思えば「みたい！」ボタンを押し、自分専用の「みたいリスト」に追加。
実際に視聴したら「みた！」ボタンで自分の履歴へ移行させます。
これにより、作品選びの悩みから解放され、アニメライフをより充実させることができます。

## サービスの差別化ポイント・推しポイント
・既存の大型レビューサイトとの差別化
大型サイトではレビューが高い注目アニメや今期のアニメが上位ランキング記事に表示されていることが多くあります。本サービスは「点数⭐️の高い順」ではなく、「見たい！」の保存された数が多い順を表示させ、ユーザーの動機にフォーカスします。
また更新が新しい作品記事を表示させる仕組みをとることで、たとえ自分が生まれていない時代の過去の名作や全く知らないアニメを、誰かが見始めて投稿すればトップに上がってきます。そして他のユーザーへの新しい作品の発見や「見たい！」を刺激してくれます。
そして、視聴前後の状態管理（見たい→みた）をワンクリックで切り替えられる点もポイントです。

・「メモする楽しさ」は残しつつ、「調べる苦労」をなくす
API連携でタイトルと画像を自動補完させることにより、ユーザーは面倒な画像探しやアップロード作業から解放され、ユーザーが一番楽しいはずの「自分の言葉で感想を書く・共有する」ことに集中できる環境を提供します。手間を「情報の検索」ではなく「愛を語る」方に使ってもらうことが狙いです。

## 使用する技術スタック
